FairStakeToken Contract API Documentation

Staking Reward Token and Governance Voting Protocol

Overview

FairStakeToken contract is an ERC20-based staking reward token that integrates staking functionality, token exchange mechanism, and governance voting interface. The contract provides the following core features:

  • Token staking and dynamic reward system
  • Fair Reward (FR) token exchange with other tokens
  • Flexible unstaking mechanism and penalty calculation
  • Modular function pause control
  • Contributor reward minting functionality
  • Governance voting interface implementation

Functions

function stake(uint256 amount) external whenModuleNotPaused(FunctionModule.STAKE) nonReentrant returns (bool)

User stakes tokens to start earning rewards.

Parameters:

  • amount - Amount of tokens to stake

Returns: bool - Whether the operation was successful

Functionality:

  • Validate staking amount is not zero
  • Check if exceeding single user maximum staking limit (30,000,000 tokens)
  • Transfer tokens from user account to contract
  • Calculate and distribute unclaimed FR rewards
  • Update user staking information and total staked amount
  • Emit Staked event
function requestUnstake(uint256 amount, uint64 waitDays) external whenModuleNotPaused(FunctionModule.UNSTAKE) nonReentrant returns (bool)

User requests to unstake staked tokens.

Parameters:

  • amount - Amount of tokens to unstake
  • waitDays - Number of days to wait (15-60 days, automatically adjusted if out of range)

Returns: bool - Whether the operation was successful

Functionality:

  • Validate unstake amount is not zero and does not exceed staked amount
  • Automatically adjust wait days to be within 15-60 days range
  • Calculate and distribute unclaimed FR rewards
  • Transfer tokens from staking state to unstake waiting state
  • Set unstake request time and wait days
  • Emit UnstakeRequested event
function unstake() external whenModuleNotPaused(FunctionModule.UNSTAKE) nonReentrant returns (bool)

User executes the actual unstake operation.

Returns: bool - Whether the operation was successful

Functionality:

  • Check if there is a pending unstake request
  • Verify lock period is completed
  • Calculate early unstake penalty (0.2% per day when wait days is less than 60)
  • Transfer remaining tokens back to user account after deducting penalty
  • Update user staking record and total unstaked amount
  • Emit Unstaked event
function claimReward() external whenModuleNotPaused(FunctionModule.REWARD) nonReentrant returns (bool)

User claims unclaimed FR rewards without unstaking.

Returns: bool - Whether the operation was successful

Functionality:

  • Calculate FR rewards earned by user since last claim
  • Validate reward amount is not zero
  • Mint FR tokens to user account
  • Update last claim time
  • Emit RewardClaimed event
function burn(uint256 amount) external whenModuleNotPaused(FunctionModule.BURN) nonReentrant

Burn FR tokens from user account.

Parameters:

  • amount - Amount of FR to burn

Functionality:

  • Validate burn amount is not zero
  • Burn specified amount of FR tokens from user account
  • Emit Burnt event
function exchangeFRForToken(address tokenAddress, uint256 frAmount) external whenModuleNotPaused(FunctionModule.EXCHANGE) nonReentrant returns (uint256)

Exchange other tokens by burning FR tokens.

Parameters:

  • tokenAddress - Address of target token to exchange
  • frAmount - Amount of FR to exchange

Returns: uint256 - Amount of tokens received from exchange

Functionality:

  • Validate target token address and FR amount
  • Check if exchange rate is set and valid
  • Verify exchange is within valid time window
  • Check if exchange amount exceeds available limit
  • Within first 48 hours, single exchange cannot exceed 1% of remaining quota
  • Burn FR tokens and transfer target tokens to user account
  • Update remaining exchange quota
  • Emit TokenExchanged event
function calculateEarnedTokens(address user) public view returns (uint256)

Calculate user's unclaimed FR reward amount.

Parameters:

  • user - User address

Returns: uint256 - Amount of unclaimed FR rewards

Functionality:

  • Get user's staking information
  • Calculate time difference since last claim (maximum 180 days)
  • Calculate reward rate based on staking amount
  • Use formula: staked amount × time × rate / (86400 × 3650000) to calculate rewards
function getRate(uint256 amount) public pure returns (uint256)

Calculate reward rate based on staking amount.

Parameters:

  • amount - Staking amount

Returns: uint256 - Reward rate

Functionality:

  • Base rate is 100 (when staking amount is less than 10,000 × 10^18)
  • When staking amount is between 10,000-20,000 × 10^18, rate equals amount divided by 10,000 × 10^18
  • When staking amount exceeds 20,000 × 10^18, rate is 200 + (amount - 20,000 × 10^18) / 40,000 × 10^18
  • Maximum rate is 500
