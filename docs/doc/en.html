FairStakeToken Contract API Documentation

Staking and Token Exchange Protocol

Overview

FairStakeToken contract is a token staking and exchange protocol based on TRON blockchain, providing the following core functionalities:

  • Token staking and reward system
  • Fair Reward (FR) and FairToken exchange mechanism
  • Early unstaking mechanism with penalty rate calculation
  • Modular function pause control
  • Multi-dimensional reward calculation and distribution

Error Types

TransferFailed()

Error thrown when token transfer fails.

InsufficientFROnChain()

Error thrown when there are not enough FR rewards available on chain.

AddressIsZero()

Error thrown when address is zero address.

AmountIsZero()

Error thrown when amount is zero.

UserNotStaked()

Error thrown when user has not staked any tokens.

UserAlreadyStaked()

Error thrown when user has already staked tokens.

UnstakeRequestNotReady()

Error thrown when unstake request is not yet ready.

UnstakeRequestAlreadyExists()

Error thrown when user already has an unstake request pending.

NoPendingUnstakeRequest()

Error thrown when there is no pending unstake request.

ExchangeLimitExceeded()

Error thrown when exchange amount exceeds the limit.

ExchangeAmountExceedsOnePercentLimit()

Error thrown when single exchange amount exceeds 1% of remaining limit within 48 hours.

ContractIsPaused()

Error thrown when contract function is paused.

OnlyOwner()

Error thrown when only contract owner can call this function.

TokenExchangeTypeInvalid()

Error thrown when token exchange type is invalid.

BurnAmountTooLarge()

Error thrown when burn amount is too large.

RewardAmountZero()

Error thrown when reward amount is zero.

NotEnoughTokensToUnstake()

Error thrown when user doesn't have enough tokens to unstake.

Events

event Staked(address indexed user, uint256 amount, uint256 rewardAmount)

Event emitted when a user successfully stakes tokens.

Parameters:

  • user - Staking user's address
  • amount - Amount of tokens staked
  • rewardAmount - FR reward amount
event UnstakeRequested(address indexed user, uint256 amount, uint256 timestamp)

Event emitted when a user successfully requests to unstake.

Parameters:

  • user - Unstaking user's address
  • amount - Amount of tokens to unstake
  • timestamp - Request timestamp
event Unstaked(address indexed user, uint256 fairAmount, uint256 fee)

Event emitted when a user successfully unstakes.

Parameters:

  • user - Unstaking user's address
  • fairAmount - FR amount received
  • fee - Early unstake penalty amount
event RewardClaimed(address indexed user, uint256 amount)

Event emitted when a user successfully claims reward.

Parameters:

  • user - Reward claiming user's address
  • amount - FR reward amount claimed
event TokenExchanged(address indexed user, uint256 frAmount, uint256 tokenAmount, uint256 timestamp)

Event emitted when a user successfully exchanges tokens.

Parameters:

  • user - Token exchanging user's address
  • frAmount - FR amount paid
  • tokenAmount - Token amount received
  • timestamp - Exchange timestamp

Functions

function stake(uint256 amount) external

Main function for users to stake tokens.

Parameters: amount - Amount of tokens to stake (in TRX units)

Operation流程：

  • Validate input parameters
  • Calculate user's FR rewards (including unclaimed rewards)
  • Process user's staking operation
  • Update staking records
  • Emit Staked event
function requestUnstake(uint256 amount) external

User requests to unstake staked tokens.

Parameters: amount - Amount of tokens to unstake

Feature特点：

  • Requires 15-60 days waiting period, specific duration calculated based on staked amount
  • Calculate user's unclaimed FR rewards
  • Set unstake request expiration time
  • Update user's staking information
  • Emit UnstakeRequested event
function unstake() external

User executes the actual unstake operation.

Feature特点：

  • Check if unstake request has expired
  • Calculate early unstake penalty rate (0.2% per day)
  • Execute token transfer
  • Update user's staking records
  • Emit Unstaked event
function claimReward() external

User claims unclaimed FR rewards.

Operation流程：

  • Calculate earned FR rewards
  • Update user's reward records
  • Emit RewardClaimed event
function exchangeFRForToken(uint256 frAmount, TokenTransferType transferType) external

Core token exchange function, allows users to exchange FR for FairToken.

Parameters:

  • frAmount - FR amount to exchange
  • transferType - Transfer type (instant or other)

Important limitations：

  • 48-hour single exchange limit: Single exchange within 48 hours not exceeding 1% of remaining limit
  • Check if contract is paused
  • Validate input parameters
  • Calculate exchange rate and exchange amount
  • Update user's exchange records
  • Execute token transfer
  • Emit TokenExchanged event
function getUserStakeInfo(address user) external view returns (StakeInfo memory)

Get user's staking information.

Parameters: user - User's address

Returns: StakeInfo - User's staking information struct

function burn(uint256 amount) external

Burn specified amount of tokens.

Parameters: amount - Amount of tokens to burn

Permissions: Only contract owner can call

function setTokenExchangeRate(uint256 rate, uint256 maxExchangeAmountPerDay, TokenTransferType transferType) external

Set token exchange rate and daily maximum exchange amount.

Parameters:

  • rate - Exchange rate (FR amount per token unit)
  • maxExchangeAmountPerDay - Daily maximum exchange amount
  • transferType - Transfer type

Permissions: Only contract owner can call
